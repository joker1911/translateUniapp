# 通话翻译助手（uni-app）

基于 uni-app 的通话翻译前端，可在通话场景下实时识别（示例数据）并调用可配置的翻译接口，附带悬浮字幕窗，方便在其他页面也能看到翻译结果。

## 功能
- 支持 Google 翻译 API、大模型接口（如 OpenAI 兼容接口）、自定义 REST API 的配置。
- 翻译目标/源语言可切换，接口地址、API Key、模型名称均可调整。
- “开始监听”模拟通话场景的识别流（示例语句），记录翻译耗时。
- 悬浮字幕窗可拖拽、随时显示/隐藏，便于跨应用查看翻译结果。
- 翻译失败时自动回退原文，避免阻塞 UI。

## 运行
1. 使用 HBuilderX 或 `@dcloudio/uni-app` CLI 导入项目。
2. 在页面内填写 API Key、接口地址等配置。
3. 点击“开始监听”体验实时翻译与悬浮字幕。
